# Wanderlust

Wanderlust is a modern travel booking web application designed to help users seamlessly plan and book their next adventure.

## ✨ Features

* **User Authentication:** Secure login, sign-up, and password recovery using Firebase Authentication.
* **Travel Booking:** Dynamic date selection, automatic duration calculation, and booking management.
* **Secure Payments:** Integrated Razorpay payment gateway for smooth and secure checkout.
* **User Profiles:** Personalized dashboard for users to view and manage their past bookings.
* **Responsive Design:** A user-friendly interface that works perfectly across all devices.

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend/Database:** Firebase
* **Payments:** Razorpay API

## 🚀 Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Harshit800-soni/Wanderlust-global.git
   ```
2. Open the project folder and run it using a local development server (e.g., Live Server extension in VS Code).
3. Ensure you have your Firebase configuration and Razorpay API keys set up in the respective JavaScript files.
